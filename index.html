Política de Privacidade - BioLearn

Última atualização: 3 de junho de 2026

Esta Política de Privacidade descreve como o aplicativo BioLearn, desenvolvido por estudantes do UNISAGRADO para fins acadêmicos e educacionais, trata as informações dos usuários. Ao utilizar este aplicativo, o usuário declara estar ciente das práticas descritas nesta Política de Privacidade.

2. Finalidade do aplicativo

O aplicativo BioLearn é um aplicativo educativo com o objetivo de ensinar conteúdos sobre animais, plantas e ecossistemas brasileiros. O aplicativo foi desenvolvido para fins acadêmicos, como parte de um projeto educacional.

3. Coleta de dados pessoais

O aplicativo BioLearn não coleta, armazena ou compartilha dados pessoais dos usuários.

O aplicativo pode utilizar apenas informações geradas durante o uso, como progresso no quiz, pontuação ou nível do usuário, armazenadas localmente no próprio dispositivo. Essas informações não são enviadas para servidores externos, banco de dados em nuvem ou terceiros.

4. Dados sensíveis

O aplicativo BioLearn não solicita, coleta ou armazena dados sensíveis dos usuários, como informações sobre saúde, religião, origem racial ou étnica, opinião política, dados biométricos ou outros dados especialmente protegidos.

5. Compartilhamento de dados

O aplicativo BioLearn não compartilha dados pessoais dos usuários com terceiros.

6. Uso de serviços de terceiros

O aplicativo BioLearn não utiliza serviços externos que coletem dados pessoais dos usuários.

7. Armazenamento e segurança dos dados

Como o aplicativo não coleta dados pessoais nem utiliza servidores externos, não há armazenamento de dados pessoais pelos desenvolvedores.

Eventuais informações de uso, como progresso, pontuação ou nível no quiz, podem ser armazenadas apenas localmente no dispositivo do usuário. Essas informações permanecem no próprio aparelho e podem ser removidas caso o usuário desinstale o aplicativo ou limpe os dados do aplicativo nas configurações do dispositivo.

8. Retenção e exclusão de dados

Como o aplicativo não coleta nem armazena dados pessoais em servidores externos, não há dados pessoais a serem excluídos pelos desenvolvedores.

Caso o usuário deseje apagar informações locais do aplicativo, poderá desinstalar o aplicativo ou limpar seus dados nas configurações do dispositivo.

9. Permissões utilizadas pelo aplicativo

O aplicativo BioLearn não solicita permissões sensíveis do dispositivo, como câmera, localização, microfone, contatos ou armazenamento externo.

Caso alguma permissão venha a ser utilizada em versões futuras, esta Política de Privacidade será atualizada para informar a finalidade da permissão solicitada.

10. Crianças e adolescentes

O aplicativo BioLearn foi desenvolvido para fins acadêmicos e educacionais. Caso seja utilizado por menores de idade, recomenda-se o acompanhamento de pais, responsáveis ou professores.

O aplicativo não tem como objetivo coletar intencionalmente dados pessoais de crianças ou adolescentes.

11. Alterações nesta Política de Privacidade

Esta Política de Privacidade poderá ser atualizada para refletir melhorias no aplicativo, mudanças legais ou ajustes nas práticas de tratamento de dados.

A data da última atualização será sempre indicada no início deste documento.
